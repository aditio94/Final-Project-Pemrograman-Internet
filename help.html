Apa tujuan MyFilesApps

Untuk Siapa MyFilesApps


Maksud dan Tujuan My-Files Membuat mahasiswa sadar diri akan pentingnya data-data pendidikan dan juga memudahkan pencarian data jika suatu saat dibutuhkan, serta mempermudah pengguna merapikan data mereka

Make students self-conscious about the importance of data education and also made it easy for the data if needed, and simplify users smoothing their data

Pengguna potensial MyFilesApps adalah kalangan mahasiswa dan dosen, dikarenakan modul-modul pembelajaran, ebook dan file-file mahasiswa sangatlah banyak dan diperlukan sebuah aplikasi untuk menampung semua file tersebut dan ada di saat dibutuhkan.
Selain Mahasiswa dan Dosen, para perlajar yang sangat gemar menyimpan file untuk kemudian dibaca kembali juga bisa memanaatkan aplikasi ini.
